Important FAQ

What is the Document Object Model (DOM)?

A tree-like structure that represents a webpage, allowing you to manipulate HTML and CSS.

How do you select an element from DOM?

Use: getElementById(), querySelector(), or querySelectorAll().

What is event delegation in the context of the DOM, and why is it useful?

Attach one listener to a parent to manage events for child elements, improving efficiency.

How do you manipulate an element's attributes and styles using the DOM?

Use setAttribute()/getAttribute() for attributes, and element.style.property for styles.
